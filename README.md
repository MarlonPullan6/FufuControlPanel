# [芙芙启动器](https://github.com/CodeCubist/FufuLauncher) 控制面板

使用FufuLauncher启动游戏注入 `DLL` 后显示的控制面板

# 构建
* 使用Visual Studio 2022&ensp;/&ensp;Visual Studio2026打开 `Client.slnx` 
* 项目配置:
    * **PlatformToolset**：`143`
    * **C++ 标准**：`std:c++17`
* 编译

# 使用
* 打开 `文档\fufu\Install` / `C:\Users\你的用户名\Documents\fufu\Install` 文件夹
* 将 `Client\x64\Release` 文件夹下的 `ControlPanel.exe` 复制到 `Install` 文件夹
* 替换掉原 `ControlPanel.exe` 即可
